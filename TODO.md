# Tarefas de Correção dos Links

## Problema
Os links usam caminhos absolutos (`/`) que não funcionam no GitHub Pages quando o site está num subdiretório (repositório).

## Solução
Adicionar a tag `<base href="../">` em todos os ficheiros HTML da pasta Html/.
